## 對同學的好處
1. 鼓勵時時學習，累積不會白費。
2. 鼓勵及早發佈，成為意見領袖。
3. 鼓勵相互參考，共同創造新意。
4. 鼓勵跨課協作，突顯個人特質。

## 為什麼是「小島」？
每個人都很渺小，就像台灣之於全世界只是一個小島一樣。與其好高騖遠望著遙不可及的夢想，不如專注當下，讓自己的小島多彩多姿，也許有一天，夢想就不再遙不可及。
