## 開源協作課程：我的履歷表
開源協作課程並不是單獨的課程，我們和大學開設的課程合作，協助同學發佈學習過程及協作成果。
* 以 GitHub 工具整理及發表成果。
* 幫助同學在修課同時累積個人履歷。
* 鼓勵跨課程間的協作。

## 課程流程 (隨合作課程調整)
1. 學期初：介紹本課程目的，向同學說明適合該課程的開源方式。
2. 學期間：（合作課程進行......）同學於線上討論開源問題。
3. 學期末：同學期末報告，本團隊提供持續開源建議。

## 對同學的好處
1. 累積履歷，學習不會白費。
2. 請互相參考，請教學相長。
3. 學會放下。
4. 跨課協作更是強者。

## 為什麼是「小島」？
每個人都很渺小，就像台灣之於全世界只是一個小島一樣。與其好高騖遠望著遙不可及的夢想，不如專注當下，讓自己的小島多彩多姿，也許有一天，夢想就不再遙不可及。
